## Down the Rabbit Hole üêá

Investigation revealed that most QR codes were generated through two primary platforms:

- **QR Code AI Art Generator** (Hugging Face Space)
- **Quick QR Art Discord Channel**

I initially struggled with the Hugging Face Space, mistakenly attempting to upload background images instead of QR codes. After reviewing the Quick QR Art documentation and studying Discord channel prompts, I successfully created multiple examples.

### Sample Creations with Prompts

**Funky:**
```
graphic design, primary colors, primitive shapes --qrw 0.95 --steps 16 --seed 7581777480381614
```

**Doggos:**
```
dogs playing in a green prairie under the sun --qrw 0.95 --steps 16 --seed 8160813035284072
```

**Mountains:**
```
mountains and trees, aesthetic, realistic --presets intricate-studio --qrw 0.9 --steps 18 --seed 5676602113908140
```

## Surfacing üåá

After experimenting with Quick QR Art's Discord bot, I generated a city skyline QR code for the website using the prompt:

```
city skyline at sunset, aesthetic, realistic --presets intricate-studio --qrw 0.95 --steps 18 --seed 6255550864009378
```

## Bonus ü§ë

I extended the QR code image using Adobe Photoshop's Generative Fill feature:

1. Created a 1280√ó720 pixel canvas
2. Centered the QR code
3. Removed edge artifacts via selection tools
4. Inverted the selection to apply generative fill with the prompt: "Extend the city without the QR code artifacts in the sky or buildings"

This technique successfully expanded the cityscape background while preserving QR code functionality, resulting in a beautiful cover image.
